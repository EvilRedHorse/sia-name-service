Sia-Name-Service

Sia-Name-Service is a Java Gradle project that provides a process to register, update, and lookup `[name].[tld]` hosts on the Sia blockchain.

Go To

Supply the skynet portal and `[name].[tld]` host that you wish to be redirected to.



Register a Host

Register a host by posting the `[name].[tld]` host that you wish to register with the skylink that it should point at and a registrant address and a fee to the register endpoint. The fee is how many siacoins to the power of ten that will be needed to register or update the linked skylink. This fee is limited to 1 on this server. To increase the fee in the future you will need to update use a different server. The registrant address should be an address that you control.




Address Bar Browser Extension

It is possible to install and configure the Redirector browser extension to add native Sia-Name-Service TLD support to your address bar. To do this first install the Firefox or Chrome browser extension, then generate a Sia-Name-Service Redirector configuration file, then navigate to the Redirector's settings, and finally import your generated configuration file.
